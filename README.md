# Flask_RESTful
Flask App working on RESTful apis and CRUD operations.

# Library to install:
<ol>
  <li>pip install flask</li>
  <li>pip install flask_restful</li>
  <li>pip install flask_jwt</li>
  <li>pip install flask_sqlalchemy</li>
</ol>

# How to run the app?
python app.py

# Endpoints for testing:
<ol>
  <li>'/store/name'</li>
  <li>'/item/name'</li>
  <li>'/item'</li>
  <li>'/items'</li>
  <li>'/stores'</li>
  <li>'/register'</li>
</ol>
